MCPE Linux Launcher
===================

This is just a fork with pre-made .sh installers to install all required libs and to compile the application.

## How to install
1. Install
```
./install_libs.sh
./install.sh
```
2. Done!

## How to run
1. Run it
```
./mcpelauncher
```
2. Done!
